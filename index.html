Three steps to autonomous work

Define, send, and let your agent handle the rest.

1

Define Your Agent

Set a system prompt, choose a model, and enable tools like code execution or file upload.

2

Send a Task

POST a task via the REST API. Attach files if needed. Get a run ID back instantly.

3

Agent Delivers Results

The agent thinks, writes code, executes, learns, and loops until the task is complete.

terminal 
$ curl -X POST https://api.orcan.dev/agents \
  -H "Content-Type: application/json" \
  -d '{
    "task": "Analyze sales.csv and create a trend chart",
    "agent": "data-analyst",
    "files": [{"name": "sales.csv", "content": "...base64..."}]
  }'

# Response:
{ "id": "a1b2c3", "status": "pending" }
